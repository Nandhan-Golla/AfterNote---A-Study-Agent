### 🚀 **How to Start Using AfterNote**

#### Quick Start (One Command)
```bash

python setup.py

# Start the full application
./start-afternote.sh

# Open in browser
# Frontend: http://localhost:3000
# Backend API: http://localhost:8000
```

#### Manual Start
```bash
# Backend only
./start-backend.sh

# Frontend only
./start-frontend.sh

# Test AI functionality
python demo_api.py
```
